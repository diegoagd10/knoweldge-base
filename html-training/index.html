Etiquetas reemplazables

Que son?

Son etiquetas que no tienen contenido, solo tienen atributos.

  • <img src="./images/profile.jpg">
  • <br>
  • <input type="datetime-local">

Ejemplo de una etiqueta reemplazable:

Que son los Atributos?

Permiten asignarle caracteristicas a la etiqueta.

Ejemplos de atributos:

  • hidden: Permite esconder el elemento.
  • width/height: Permite controlar el tamano del elemento.
  • title: Permite agregar un tooltip a la imagen cuando el cursor esta por encima de la imagen.
  • alt: Permite poner un texto alternativo cuando la imagen no se cargo.
  • src: Permite cargar una imagen/video/audio/url dependiendo de la etiqueta.
  • ref: [noreferrer] Evita que el navegador envie informacion de la pagina de origen a la pagina destino.
  • role Permite agregar un rol semantico a una etiqueta.
NOTE:

No todos los atributos estan disponibles para todas las etiquetas. Hay algunas etiquetas que son exclusivas para ciertos elementos.

El HTML semantico

Que significa la palabra Semantico?

Que tiene un significado.

HTML semantico

Describe el contenido del documento ayudando a indificar que es cada cosa en el documento.

Ejemplos:

  • La etiqueta p indica que se esta describiendo un parafo.
  • La etiqueta img indica que se esta describiendo una imagen.
  • La etiqueta h1 indica que se esta describiendo un titulo.
  • La etiqueta aside, ayuda a identificar contenido que esta relacionado de manera indirecta con el contenido principal.
  • La etiqueta section agrupa diferentes bloques de contenido que estan relacionados entre si.
  • La etiqueta article agrupa contenido que tiene sentido por si solo (Puede ser exportado a otro sitio y seguira teniendo sentido).
  • La etiqueta footer contiene detalles finales relacionados al contenido precedente.
  • La etiqueta small es usada para mostrar informacion secundaria como: Copyright y texto legal.
  • La etiqueta nav permite crear un seccion para navegar en el sitio web o documento.

Atributos especiales de la etiqueta <ol>

type

Permite cambiar el formato de la lista a:

  • Numeros.
  • Letras mayusculas.
  • Letras minusculas.
  • Entre otros.

Ejemplos:

Numeros
  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
Letras mayusculas
  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
Letras minusculas
  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

reversed

Permite invertir el orden de la lista.

Ejemplo:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Atributo value para la etiqueta <li>

Permite cambiar la secuencia de la lista

Ejemplo:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. Cuarto elemento
  5. Quinto elemento
  6. Sexto elemento
  7. Septimo elemento
